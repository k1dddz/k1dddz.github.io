在linux系统中搭建git仓库

    本文的场景为在本地虚拟机中搭建git仓库，以此管理主机中的项目。

环境说明

  • 虚拟机 : VM

  • 操作系统 ：

    • 主机：windows10

    • 虚拟机：ubuntu16.04

  • 连接协议：SSH

  • 软件：git、github desktop(可选)

操作步骤

一、前置准备

  1. 在[git官网](Git - Downloading Package)中下载windows版本的git

  2. 在vm虚拟机中安装unbuntu操作系统

  3. 为虚拟机系统设置一个固定ip（如：192.168.56.101）

二、在 Ubuntu 虚拟机上安装并配置 Git 服务

  1. 安装git

  2. 创建 Git 用户（用于管理仓库）
